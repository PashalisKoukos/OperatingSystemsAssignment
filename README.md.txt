ΟΔΗΓΊΕΣ ΕΚΤΈΛΕΣΗΣ ΓΙΑ ΟΛΑ ΤΑ ΕΚΤΕΛΕΣΙΜΑ ΑΡΧΕΊΑ.
Για τα υπόλοιπα μέρη έχω πλήρη υλοποίηση και εξήγηση στο pdf template.

-> ΜΈΡΟΣ Ε :
Μπαίνουμε στον φάκελο monitor και μεταγλωττίζουμε το αρχείο με :
gcc -o analyze_log analyze.c

Εκτελούμε με :
./analyze_log dir_name/file_name.log 

Στο file_name αντικαθιστώ με το κατάλληλο όνομα αρχείου log που υπάρχει μέσα στον κατάλογο dir_name
(π.χ. system.log ή network.log ή security.log)

Στο dir_name αντικαθιστώ με όνομα φακέλου που περιέχει log αρχεία μέσα για να πάρω ένα (π.χ raw).

================================================================================================================
-> ΜΈΡΟΣ ΣΤ :
Για το shell script μπαίνουμε στον φάκελο monitor κι αν δεν έχει δικαίωμα εκτέλεσης το .sh (ls -al για να δούμε)
δίνω με chmod +x run_monitor.sh και στη συνέχεια εκτελώ με :

./run_monitor dir_name 

Στο dir_name αντικαθιστώ με όνομα φακέλου που έχει μέσα log αρχεία (π.χ. raw στη περίπτωσή μας)
=================================================================================================================
-> ΜΈΡΟΣ Ζ : Μπαίνουμε στον φάκελο monitor και μεταγλωττίζουμε το αρχείο με :
gcc parallel_analyze.c -o parallel_analyze

Εκτελούμε με :
./parallel_analyze dir_name/file_name1.log dir_name/file_name2.log ... dir_name/file_nameN.log 

Στο file_name αντικαθιστώ με το κατάλληλο όνομα αρχείου log που υπάρχει μέσα στον κατάλογο dir_name
(π.χ. system.log ή network.log ή security.log) έχοντας τη δυνατότητα να βάλω παραπάνω από 1 αρχεία. 
Αυτή είναι η παράλληλη ανάλυση με threads.

Στο dir_name αντικαθιστώ με όνομα φακέλου που περιέχει log αρχεία μέσα για να πάρω όσα θέλω (π.χ raw).
Φυσικά μπορώ να επιλέξω απο διαφορετικούς φακέλους αρχεία log.
